Goose Game

A Unity-based arcade game featuring flight mechanics and energy management, developed through a comprehensive design thinking process

Unity Arcade Flight Design Thinking

Project Overview

Goose Game is an arcade-style flying game developed in Unity as part of a design thinking process exploration. Players control a goose navigating through various obstacles while managing energy levels and mastering flight mechanics.

The project served as both a game development exercise and a case study in iterative design methodology. Through multiple development cycles, the game evolved from a simple flight concept to a polished arcade experience with refined mechanics and engaging progression systems.

Core Mechanics

Flight Controls

Intuitive flying mechanics with physics-based movement, allowing players to soar, glide, and navigate with precision.

Energy Management

Strategic energy system requiring players to balance active flying with gliding to maintain altitude and progress.

Progressive Difficulty

Increasingly challenging obstacles and environmental hazards that test player skill and adaptation.

Responsive Controls

Tight, responsive input system optimized for both keyboard and controller gameplay.

Design Thinking Process

The development followed a structured design thinking methodology:

Problem Identification

Identified the need for engaging flight mechanics in arcade games and explored player motivation in simple game loops.

Ideation & Concept

Brainstormed various flight-based game concepts, settling on energy management as a core challenge element.

Prototyping

Rapid prototyping in Unity to test basic flight mechanics and energy systems with immediate feedback.

Testing & Iteration

Multiple testing cycles with user feedback leading to refined controls, balanced energy consumption, and improved visual feedback.

Polish & Presentation

Final implementation with polished visuals, sound integration, and comprehensive documentation of the design process.

Technical Implementation

The game showcases various Unity development techniques:

Physics Integration

Custom physics implementation for realistic flight mechanics with adjustable gravity and drag systems.

UI Systems

Comprehensive user interface including energy bars, tutorial screens, and menu systems with consistent visual design.

Animation Pipeline

Sprite-based animation system for the goose character with multiple movement states and transitions.

Scene Management

Efficient scene loading and management system supporting multiple game states and smooth transitions.

Development Iterations

The project went through multiple development builds:

  • Base Version: Core flight mechanics and basic obstacle avoidance
  • Enhanced Flying: Improved physics and more responsive controls
  • New Assets: Updated sprites, backgrounds, and visual elements
  • Menu System: Complete UI overhaul with tutorials and help screens
  • New Background: Enhanced environmental art and atmospheric elements
  • Final Polish: Balanced gameplay, refined mechanics, and bug fixes

Key Features

  • Smooth, physics-based flight mechanics
  • Strategic energy management system
  • Progressive difficulty with varied challenges
  • Comprehensive tutorial and help systems
  • Custom sprite animations and visual effects
  • Responsive control system for multiple input types
  • Complete menu and UI system
  • Available on itch.io with full documentation

Project Details
